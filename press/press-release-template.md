### What is DoPlan?

DoPlan automates the tedious aspects of project management by generating documentation, creating project structures, managing Git workflows, tracking progress, and providing visual dashboards. It integrates seamlessly with popular AI-powered IDEs including Cursor, Claude CLI, Gemini CLI, Codex CLI, OpenCode, and Qwen Code.

### Key Features

- **Automated Project Setup**: Initialize projects with structured workflow in seconds
- **Multi-IDE Integration**: Works with 6 different AI-powered development environments
- **Visual Dashboards**: Real-time progress monitoring with interactive TUI
- **GitHub Automation**: Automatic branching, PR creation, and commit tracking
- **Template System**: Customizable templates for plans, designs, and tasks
- **Checkpoint System**: Create and restore project snapshots at any time

### Technical Details

Built with Go and featuring a fullscreen TUI powered by Bubbletea, DoPlan provides:
- Command-line interface for all operations
- Integration with Git and GitHub CLI
- Template-based document generation
- JSON-based state management
- Support for multiple IDE command formats

### About

DoPlan is an open-source project built with Go, designed to help developers maintain organized, well-documented projects throughout the development lifecycle. It combines automated workflow management with seamless IDE integration to reduce project management overhead.
