# DoPlan CLI - One Pager

## What is DoPlan?

DoPlan is a project workflow automation tool that transforms app ideas into structured, documented, and trackable development projects. It integrates seamlessly with AI-powered IDEs and automates the entire development lifecycle.

## Key Features

✅ **Automated Workflow** - From idea discussion to deployment  
✅ **Multi-IDE Support** - Works with Cursor, Claude, Gemini, Codex, OpenCode, Qwen  
✅ **Visual Dashboards** - Real-time progress tracking (Markdown & HTML)  
✅ **GitHub Automation** - Auto-branching, PR creation, commit tracking  
✅ **Template System** - Customizable document templates  
✅ **Checkpoint System** - Time Machine for project snapshots  
✅ **Fullscreen TUI** - Beautiful interactive terminal interface

## Installation

```bash
# Homebrew (macOS)
brew install doplan

# npm (Node.js)
npm install -g @doplan-dev/doplan-cli

# Direct download
# Visit https://github.com/DoPlan-dev/CLI/releases
```

## Quick Start

```bash
# 1. Install in your project
doplan install

# 2. Select your IDE
# Choose from: Cursor, Gemini CLI, Claude CLI, Codex CLI, OpenCode, Qwen Code

# 3. Start using commands
# In your IDE, type: /Discuss
```

## Workflow

1. **Discuss** - Refine your app idea with AI assistance
2. **Generate** - Create PRD, API contracts, and project structure
3. **Plan** - Break down into phases and features
4. **Implement** - Develop with automated Git workflow
5. **Track** - Monitor progress with visual dashboards
6. **Deploy** - Complete features and merge to main
