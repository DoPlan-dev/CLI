## Installation

```bash

go install github.com/DoPlan-dev/CLI/cmd/doplan@v0.0.18-beta

# Or build from source
git clone https://github.com/DoPlan-dev/CLI.git
cd CLI/cli
git checkout v0.0.18-beta
go build -o doplan ./cmd/doplan
```
